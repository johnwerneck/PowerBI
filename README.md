PBH_CadastroUnico é um projeto de análise de dados em Power BI utilizando como base de dados um arquivo .csv contendo informações da população cadastradas no Cadastro Único em Belo Horizonte.
Esses dados foram retirados diretamente do portal da Prefeitura de Belo Horizonte através do link https://dados.pbh.gov.br/dataset/pessoas-no-cadastro-unico (arquivo 02/2026- PESSOAS NO CADASTRO ÚNICO). 
Link da base de dados (baixar .csv): https://dados.pbh.gov.br/dataset/pessoas-no-cadastro-unico/resource/9a678202-f795-4bef-afa2-993aa9fb1029 (último acesso em 2026/04/24).

Nesta pasta há 3 arquivos: a base de dados, o arquivo do Power BI e um arquivo .pdf com informações de referência sobre os dados da tabela. 
Esse último arquivo também é original da prefeitura podendo ser obtido no primeiro link, no arquivo "Dicionário de Dados de PESSOAS NO CADASTRO ÚNICO".

OBJETIVO
  Analisar a quantidade e o perfil das pessoas no Cadastro Único em Belo Horizonte até fevereiro de 2026.

MÉTODO/MODELAGEM
  Modelo Estrela: A tabela original contendo todos os dados foi segmetada em tabelas menores, tornando a tabela orinal na tabela fato e as outras representando as dimensões.
